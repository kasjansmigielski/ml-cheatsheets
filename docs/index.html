- **Wyjaśnienie algorytmu** - jak działa i dlaczego

- **Kluczowe parametry** - co można dostroić i jak

## 📋 Jak korzystać z ML Cheatsheets?

**Najlepsza metoda nauki:**

1. **Przeczytaj teorię** - zrozum podstawy algorytmu
